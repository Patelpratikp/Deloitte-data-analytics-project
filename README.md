## 📊 Project 1: Machine Downtime Analysis (Tableau)

### 🔹 Objective
To analyze telemetry data from multiple factories and identify:
- Which factory had the highest machine downtime
- Which machine types contributed most to failures

### 🔹 Tools Used
- Tableau

### 🔹 Key Steps
- Data import and schema selection
- Created calculated field "Unhealthy" to measure downtime
- Built bar charts:
  - Down Time per Factory
  - Down Time per Device Type
- Created interactive dashboard with filters

### 🔹 Key Insights
- Identified factory with highest downtime
- Laser-based machines showed highest failure rates

## 📈 Project 2: Gender Pay Equality Analysis (Excel)

### 🔹 Objective
To evaluate gender pay equality across different job roles and factories.

### 🔹 Tools Used
- Microsoft Excel

### 🔹 Key Steps
- Created a new column "Equality Class"
- Used formula:=IF(ABS(C2)<=10,"Fair",IF(ABS(C2)<=20,"Unfair","Highly Discriminative"))
- - Classified equality scores into:
- Fair
- Unfair
- Highly Discriminative

### 🔹 Key Insights
- Senior roles (C-Level, VP, Manager) show higher inequality
- Multiple factories have highly discriminative pay gaps
- Technical roles are relatively more balanced
